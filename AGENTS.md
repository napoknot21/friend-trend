# Repository Instructions

## Python Style
- Add `from __future__ import annotations` at the top of Python files.
- When editing Python, prefer the local house style:
  ```python
  def func_name (arg : dtype) :
      """
      Brief description
      """
  ```
- Keep spaces around parentheses and before `:`, matching the user's preferred style.
- Prefer triple-double-quoted docstrings directly under the function definition.

## Notes
- Do not reformat large areas of Python code unless the user explicitly asks for a repo-wide style pass.
